Crank up! is a 2D mobilegame for android that was made as a school project for a team called Glitching Wizards.
The game is free and can be downloaded from our website https://webpages.tuni.fi/24tiko2a/
The objective of the game is to fill the placement area in each level with the blocks you are given. You get points depending on how much of the area you fill.
The game was made using Godot
